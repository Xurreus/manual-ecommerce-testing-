# Test Plan Ecommerce Website

## 1. Introduction
This document defines the testing objectives, scope, approach, and resources for the ecommerce website.

## 2. Scope
Functional Testing:
- Registration
- Login
- Search
- Product Page
- Cart
- Checkout

Non-Functional Testing:
- Usability
- UI consistency
- Basic performance checks

## 3. Testing Types
- Smoke Testing
- Functional Testing
- Regression Testing
- Exploratory Testing

## 4. Test Environment
- Browser: Chrome 128+
- OS: Windows / Android
- Test data will include 5 user accounts

## 5. Deliverables
- Test Cases
- Bug Reports
- Daily test summary
